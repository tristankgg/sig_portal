# Signature Appliances Rewards Portal

A professional web portal for managing customer referrals under the Signature Appliances rewards program.

## Features

- **Referral Management**: Add, view, and manage customer referrals
- **User Authentication**: Dashboard shows logged-in user information
- **Responsive Design**: Professional sidebar navigation with account and settings sections
- **Form Validation**: Required fields for customer details
- **Status Tracking**: Monitor referral status (pending, active)
- **Clean UI**: Modern interface inspired by professional project management tools

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/tristankgg/sig_portal.git
   ```

2. Open `index.html` in your web browser
   ```bash
   open index.html
   ```

## How to Use

1. Click the **+ Add Referral** button to open the referral form
2. Fill in the required customer details:
   - Customer name
   - Phone number
   - Business name
3. Optionally add email and notes
4. Click "Add referral" to submit
5. View all referrals in the table below
6. Use the edit (✎) and delete (🗑) buttons to manage entries

## Referral Form Fields

- **Customer Name** (required): Full name of the customer
- **Phone Number** (required): Contact phone number
- **Business** (required): Customer's business name
- **Email** (optional): Customer's email address
- **Notes** (optional): Additional information
